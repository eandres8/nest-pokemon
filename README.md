# Develop environment

1. clone repository
2. Run
````
yarn install
````
3. If Nest CLI is not installed, run:
```
npm i -g @nestjs/cli
```
4. run database
```
docker-compose up -d
```

## Stack for this application
* MongoDB
* NestJS
* Typescript
